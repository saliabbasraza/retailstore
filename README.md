#Instructions

##Build
- Run ``mvn clean verify``

##Run
- Run jar file ``java -jar target/retailstore-1.0-SNAPSHOT-jar-with-dependencies.jar ``

##Jococo Test Coverage
- Open ``target/site/jacoco/index.html`` in browser to see complete coverage report.

##PMD
- Run ``mvn pmd:check`` 
<br>It will check for basic code smells.
